FAQ

What's the difference between Registry and Server?​

Both implementations allow you to access the functionalities of Hub. The biggest difference is how they store data. Registry will assign unique identifiers to datasets when you create one. A supporting database that maps unique identifiers to datasets is therefore also used. Server stores everything on the file system. Check them both and see which one suits your needs best! If you're undecided, use Registry.
