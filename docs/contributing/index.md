# Contributing

Thank you for your interest in contributing to DroneDB!

## Ways to Contribute

- **Build from source** - Develop and test new features
- **Report issues** - Help us improve by reporting bugs
- **Documentation** - Improve or expand the docs
- **Code contributions** - Submit pull requests

## Building from Source

- See [Building from Source](contributing/building-from-source) for detailed instructions on building DroneDB on Windows and Linux.

## Repositories

| Repository | Description |
|------------|-------------|
| [DroneDB/DroneDB](https://github.com/DroneDB/DroneDB) | Core C++ library and CLI |
| [DroneDB/Registry](https://github.com/DroneDB/Registry) | Web platform backend and frontend |
| [DroneDB/docs](https://github.com/DroneDB/docs) | This documentation |
