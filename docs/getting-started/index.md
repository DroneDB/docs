# Getting Started

Welcome to DroneDB! This guide will help you get up and running quickly.

## What is DroneDB?

DroneDB is a comprehensive set of tools to inspect, manage, and share geospatial data. It provides a complete solution for organizing and distributing images, orthophotos, digital elevation models, point clouds, 3D models, and vector files.

## Quick Start Options

Depending on your needs, there are several ways to get started:

### Option 1: Use DroneDB Hub (Recommended)

The easiest way to start is with [DroneDB Hub](https://hub.dronedb.app), our hosted Registry platform:

1. [Register an account](https://dronedb.app/register) (it's free)
2. Upload your first dataset
3. Explore, share, and collaborate

### Option 2: Self-host Registry

For full control over your data, self-host your own Registry instance:

- See [Registry Getting Started](registry/getting-started) for Docker and native installation

### Option 3: Use the CLI

For local data management and automation, use the `ddb` command-line tool:

- See [CLI Installation](/docs/getting-started/installation#cli-installation) for setup instructions
